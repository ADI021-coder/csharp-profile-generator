# Personal Profile Generator (C#)

A console application that generates a basic personal profile based on user input such as name, year of birth, height, and city.

## Features
- Dynamic user input collection
- Age calculation logic
- Unit conversion for height
- Structured output formatting

## Concepts Used
- Variables and data types
- Type casting
- Arithmetic operations
- Console input/output

## Notes
This project focuses on building complete, small-scale C# programs using core language fundamentals.
